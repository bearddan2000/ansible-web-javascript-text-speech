# ansible-web-javascript-text-speech

## Description
A small project to deminstrate a browser
feature that turns text to speach. Ansible
is used to install apache server.

## Tech stack
- ansible
- jquary 3.5

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
